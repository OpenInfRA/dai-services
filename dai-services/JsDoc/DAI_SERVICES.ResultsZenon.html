Class: ResultsZenon

DAI_SERVICES. ResultsZenon

Constructor

new ResultsZenon()

This class contains only static members and methods which refer to Zenon results retrieved by queries. It is recommended to use these members and methods in a static way.
Zenon is a library catalog for archaeological research.

(static) appendZenonResult(entry)

This function appends a JSON entry as table row to the result table. If the title and the authors are null, this entry will not be displayed. You should override this function if you would like to use another presentation of the data (e.g. div container than table layout).
