DAI_SERVICES. ResultsGazett

Constructor

new ResultsGazett()

This class contains only static members and methods which refer to Gazetteer results retrieved by queries. It is recommended to use these members and methods in a static way.
Gazetteer is a geographical dictionary. Usually, a Gazetteer also provides a map. In this case the map is not implemented since this implementation focusses on the search results and integration into another website. If the map feature is required it is recommented to use the original Gezetteer website and to copy and pasete search results (URL or id).

(static) getParent(parentId) → {string}

This function implements another ajax query in order to retrieve the parent of a specific geographical location. This is done because there is only the parent id as information about the parant in the result. In order to retrieve full information about the parent it is necessary to query the Gazetteer service again with this id.
