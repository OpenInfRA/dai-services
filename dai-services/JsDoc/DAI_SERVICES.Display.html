Class: Display

DAI_SERVICES. Display

Constructor

new Display()

This class contains only static members and methods. It is a display utility which collects all functions and variables used to control the display (e.g. of display components and strings). Override these functions to achieve different behaviors. It is recommended to use these members and methods in a static way.
